# MediaVault

MediaVault is a full-stack web application for managing bookmarks, articles, videos, and other media in one place.

This project is being developed as a professional portfolio application using modern web technologies and best development practices.

---

## 🚀 Tech Stack

### Frontend
- Vue 3
- Vite
- Axios

### Backend
- Node.js
- Express

### Database
- PostgreSQL
- Prisma (coming soon)

### DevOps
- Docker
- Docker Compose
- Git
- GitHub

## ▶️ Running the Project

Clone the repository:

```bash
git clone git@github.com:Bobcicle/mediavault.git
```

Start the application:

```bash
docker compose up --build
```

Frontend:

```
http://localhost:5173
```

Backend:

```
http://localhost:3000
```

pgAdmin:

```
http://localhost:5050
```
