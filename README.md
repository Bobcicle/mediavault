# MediaVault

MediaVault is a full-stack media management application built with:

- Vue 3
- Express
- PostgreSQL
- Prisma
- Docker

## Features

- Bookmark management
- Categories
- Search
- Favorites
- User authentication

## Project Structure

- client/ - Vue frontend
- server/ - Express backend
- database/ - Database files
- docker/ - Docker configuration
- docs/ - Project documentation
